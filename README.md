# Permapages Widget Template

This template uses Typescript and Preact to build a Permapage Widget!

## Usage

```sh
npx degit twilson63/widget-template my-project
cd my-project
```

## Build Widget

```sh
npm run build
```

## Run Example Server

```sh
npm run example
```

## Deploy to Arweave

```sh
npm run deploy [path to wallet.json]
```
